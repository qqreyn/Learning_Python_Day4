# Learning_Python_Day4

Today's topic is conditional statements!

**Goal:** Write programs that make decisions using if, elif, and else statements

- **task1:** Check if a number is positive, negative, or zero

- **task2:** Compare two numbers and print which one is larger

- **task3:** Check if a number is even or odd

- **task4:** Grade calculator: convert score to letter grade (A, B, C, D, F)

- **task5:** Age checker: determine if someone is a child, teen, adult, or senior

**mini-project:**
Password Validator: Ask the user for a password and check if it meets the requirements (minimum 8 characters and contains at least one number). Give specific feedback about what is wrong or confirm if the password is valid.
